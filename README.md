# REEL DREAD OFFLINE

This version requires **no sign-in, no API key, and no account**.

## What it is
A single self-contained HTML movie picker app with a built-in curated catalog.

## How to use
1. Upload `reel-dread-offline.html` to your GitHub repo, or open it directly in a browser.
2. On GitHub Pages, publish from the main branch root.
3. Open the page on your phone and save it to your home screen.

## What it includes
- Genre -> sub-genre flow
- 5 movie picks at a time
- New 5 button for different titles
- Score and year filters
- Watchlist
- Dark horror theme
- Original in-app poster styling instead of copyrighted poster images

## Limitation
Because there is no API, the catalog is curated and finite rather than live.
